Créer un compte Transen

Le nom ne doit être composé que de lettres. Le nom de famille est requis.
Le prénom ne doit être composé que de lettres. Le prénom est requis.
Le numéro est trop court. Le numéro est trop long. Le numéro de téléphone est requis.
La date de naissance est requise. Vous devez avoir au moins 18 ans.
Le type de compte est requis.
Le mot de passe doit contenir au moins une majuscule, une minuscule, un chiffre et un caractère spécial. Le mot de passe est trop court. Le mot de passe est trop long. Le mot de passe est requis.
